# Пользовательские истории

## История 1: Персональный план обязательных прививок для кота

**Как** владелец кота  
**Я хочу** добавить кота (дата рождения) и получить план обязательных прививок с датами  
**Чтобы** я мог(ла) вести вакцинацию в одном месте и не пропускать сроки.

## История 3: Просмотр ближайших прививок и отметка статуса

**Как** владелец кота или собаки  
**Я хочу** видеть список ближайших запланированных прививок и отмечать выполненные  
**Чтобы** не пропускать даты и понимать актуальный статус вакцинации для каждой прививки.

---

## История 4: Учет нескольких питомцев в одном приложении

**Как** владелец нескольких питомцев  
**Я хочу** хранить отдельные планы прививок для каждого питомца и переключаться между ними  
**Чтобы** управлять вакцинацией всех питомцев без путаницы и повторных записей.
