# Example Mapping

## Когда использовать

Используй этот скилл, когда нужно:
- уточнить размытые требования перед разработкой;
- синхронизировать понимание между бизнесом, аналитикой и разработкой;
- подготовить качественные acceptance criteria и тест-кейсы.

## Суть метода

Example Mapping помогает разложить функциональность на 4 типа карточек:
- **Story** — пользовательская история или сценарий;
- **Rules** — бизнес-правила и ограничения;
- **Examples** — конкретные примеры поведения системы;
- **Questions** — открытые вопросы и неопределенности.

## Правила проведения

1. Начинай с одной истории за раз.
2. Формулируй правила независимо от реализации.
3. Делай примеры максимально конкретными (входные данные -> ожидаемый результат).
4. Любая неясность сразу попадает в Questions.
5. Если вопросов слишком много, дроби историю.

## Пошаговый workflow

1. Зафиксируй Story:
   - роль;
   - действие;
   - ценность.
2. Выдели Rules:
   - обязательные условия;
   - ограничения;
   - исключения.
3. Добавь Examples для каждого правила:
   - позитивные сценарии;
   - граничные случаи;
   - негативные сценарии.
4. Собери Questions:
   - что не определено;
   - какие решения нужны от бизнеса/продукта;
   - какие данные отсутствуют.
5. Сформируй итог:
   - что готово к разработке;
   - что требует уточнений;
   - какие задачи на следующий шаг.

## Формат результата

```markdown
# Example Mapping: [Название истории]

## Story
Как [роль]
Я хочу [действие]
Чтобы [ценность]

## Rules
1. Rule 1: ...
   - Пример A: [условия] -> [ожидаемый результат]
   - Пример B: [условия] -> [ожидаемый результат]
2. Rule 2: ...
   - Пример A: [условия] -> [ожидаемый результат]

## Questions
1. ...
2. ...

## Готовность
- Готово к разработке: [да/нет]
- Что нужно уточнить: ...
- Следующее действие: ...
```

## Критерии качества

Перед завершением проверь:
- каждое правило подтверждено хотя бы одним примером;
- есть граничные и негативные сценарии;
- вопросы действительно открытые и требуют решения;
- итог позволяет принять решение о старте разработки.
