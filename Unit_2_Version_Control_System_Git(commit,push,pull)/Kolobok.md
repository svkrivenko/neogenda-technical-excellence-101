Жили-были Старик со Старухой. Однажды Старик рано утром попросил Старуху испечь Колобка.

## 2. Как Колобок убежал
Открыла Старуха окно, чтобы приманить соседей свежевыпеченным пирогом, а Колобок и был таков.
