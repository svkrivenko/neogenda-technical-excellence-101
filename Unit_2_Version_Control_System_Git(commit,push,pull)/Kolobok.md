# Сказка про колобка

## 1. Как Колобок появился
Жили-были Старик со Старухой. Однажды Старик рано утром попросил Старуху испечь Колобка.
Пошла Старуха в чатжпт искать рецепт - не нашла. Пошла она в дип сик - та же фигня.
Вдруг видит - вывалилась из ящика старая дискета, а на ней шикарный рецепт, проверенный временем.
